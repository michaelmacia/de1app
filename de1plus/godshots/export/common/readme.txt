God shots are autonatically exported as "common" espresso data CSV "EEX format" files here.

Programs such as the Smart Espresso Profiler https://www.naked-portafilter.com/smart-espresso-profiler/ can import this format.

Exported files are never auto-deleted.  

If you delete all the files in this directory, the next time you run the tablet software, your existing God shots will automatically be exported here.

Note that this EEX data format was created as a collaboration between Decent Espresso and Smart Espresso Profiler.
